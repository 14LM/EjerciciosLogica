ENUNCIADO

Crea un programa JavaScript que se ejecute en el navegador. El programa pedirá introducir un no desde una ventana de prompt y calculará el factorial de dicho no. El factorial de un no se representa matemáticamente con el símbolo! y es el resultado de multiplicar ese no por todos los que le preceden. Por ejemplo 5!=5x4x3x2x1, es decir, 5!=120. El programa debe utilizar un bucle for para hallar el resultado y mostrar este en el documento web.
